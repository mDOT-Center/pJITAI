Intervention Settings / Update Point

  • Explanation
  • HeartSteps Example

At each update point, for each individual, the personalization method is updated. (In other words, the parameters in the Bayesian linear regression model for the proximal outcome are updated.)

To choose the update point, you can consider choosing it to be different from the decision points so that by the update point, all the data from the previous decision points have been uploaded to the cloud.

In HeartSteps, the personalizing JITAI updated the parameters each day at midnight because midnight is not close to any of the five decision points. If a temporary wifi issue occurs when data is uploaded to the cloud from a mobile device, it provides enough time for the data to be re-uploaded if needed.
